# TravelPlanner — Планировщик путешествий

**TravelPlanner** — это веб-приложение, разработанное на Python с использованием фреймворка Flask. Проект помогает пользователям создавать личные маршруты поездок, добавлять места с автоматическим определением координат (геокодирование) и просматривать их на интерактивной карте.

## Ключевые компоненты

### Основной функционал
- Регистрация и авторизация пользователей
- Создание, просмотр, редактирование и удаление поездок
- Добавление мест в поездку с указанием адреса (автоматическое получение координат через API Яндекс.Карт)
- Интерактивная карта с отметками всех мест поездки
- Адаптивный интерфейс на Bootstrap

### Техническая реализация

#### Backend
- Фреймворк **Flask**
- ORM **SQLAlchemy** для работы с базой данных
- Система аутентификации **Flask-Login**
- Обработка форм с помощью **Flask-WTF**

#### Frontend
- Шаблонизатор **Jinja2**
- Стилизация с **Bootstrap 5**
- Интерактивные формы и карта (JavaScript)

#### Интеграции
- **API Яндекс.Карт** (геокодер для преобразования адресов в координаты)
- Отображение карты с метками через JavaScript API

### Особенности реализации
- Двухэтапная проверка пароля при регистрации
- Валидация уникальности email
- Геокодирование адреса с обработкой ошибок (если адрес не найден, показывается сообщение)
- Привязка всех данных к конкретному пользователю через внешние ключи
- Модульная архитектура: модели, формы и маршруты вынесены в отдельные файлы
- Использование переменных окружения для конфиденциальных данных (ключ API, секретный ключ)

### Технологический стек
- **Язык программирования**: Python 3.9+
- **Фреймворки**: Flask, Flask-Login, Flask-WTF, Flask-SQLAlchemy
- **База данных**: SQLite (в разработке), возможна замена на PostgreSQL
- **Внешние API**: Яндекс.Карты (Geocoder)
- **Фронтенд**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Дополнительные библиотеки**: requests, python-dotenv

## Установка и запуск

### Требования
- Python 3.9 или выше
- pip
- (рекомендуется) виртуальное окружение
